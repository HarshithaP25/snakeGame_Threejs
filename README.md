# 3D Nokia Snake Game (Three.js)

Retro 3D snake game using Three.js. Open `index.html` to play.

## Controls
- Move: Arrow keys or W A S D
- Speed Up / Down: buttons on bottom-left
- Restart: Play Again button on Game Over screen
